# Nodejs Api Project

## Requirement
- Nodejs 6.x
- Express 4.x
- Mysql 5.x

## Setup

- Install npm
```
npm init

npm install
```

- Install express
```
npm install express --save

npm install express-generator -g

express -h

express --view=pug
```

- Run the app with command

On MacOS or Linux, run the app with this command:
```
DEBUG=myapp:* npm start
```
On Windows, use this command:
```
set DEBUG=myapp:* & npm start
```
## Running Test

Access the app

```
http://localhost:3000/
```
